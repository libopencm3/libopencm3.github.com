Detailed Description

  • E_BUFF: External pin with buffer enabled
  • EP_BUFF: External pin and on-chip peripherals with buffer
  • E: External pin without buffer
  • EP: External pin and on-chip peripherals without buffer
  • SH_E_BUFF: Sample & Hold, External pin with buffer enabled
  • SH_EP_BUFF: Sample & Hold, External pin and on-chip peripherals with buffer
  • SH_E: Sample & Hold, External pin without buffer
  • SH_EP: Sample & Hold, External pin and on-chip peripherals without buffer
